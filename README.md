# Full Stack Todo Application (MERN)

A full stack Todo management application built using React, Node.js, Express.js, and MongoDB.

## Features
- Add Todos
- Delete Todos
- Toggle Completion Status
- Persistent Database Storage
- REST API Integration

## Tech Stack
Frontend: React.js, Axios, Tailwind CSS  
Backend: Node.js, Express.js  
Database: MongoDB (Mongoose)

## Installation

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend/todo
npm install
npm run dev
